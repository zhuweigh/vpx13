nscapacity module¶

class nscapacity¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for capacity resource.

bandwidth¶

System bandwidth limit.

platform¶

appliance platform type.<br/>Possible values = VS10, VE10, VP10, VS25, VE25, VP25, VS200, VE200, VP200, VS1000, VE1000, VP1000, VS3000, VE3000, VP3000, VS5000, VE5000, VP5000, VS8000, VE8000, VP8000, VS10000, VE10000, VP10000, VS15000, VE15000, VP15000, VS25000, VE25000, VP25000, VS40000, VE40000, VP40000, VS100000, VE100000, VP100000, CP1000.

vcpu¶

licensed using vcpu pool.

edition¶

Product edition.<br/>Possible values = Standard, Enterprise, Platinum.

unit¶

Bandwidth unit.<br/>Possible values = Gbps, Mbps.

nodeid¶

Unique number that identifies the cluster node.<br/>Maximum length = 31.

actualbandwidth¶

Bandwith in MBPS.

vcpucount¶

number of vCPUs licensed.<br/>Minimum value = 1<br/>Maximum value = 20.

maxvcpucount¶

number of max vCPUs.<br/>Minimum value = 1<br/>Maximum value = 20.

maxbandwidth¶

Maximum Bandwidth.

minbandwidth¶

Minimum Bandwidth.

instancecount¶

VPX will consume one instance and MPX will consume zero instance.
