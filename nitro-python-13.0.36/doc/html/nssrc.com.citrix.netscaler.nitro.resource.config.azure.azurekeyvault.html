azurekeyvault module¶

class azurekeyvault¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for Azure Key Vault entity resource.

name¶

Name of keyvault entity in NetScaler.<br/>Minimum length = 1.

azurevaultname¶

Name of the Azure vault account as configured in azure portal.<br/>Minimum length = 3<br/>Maximum length = 25.

azureapplication¶

Name of Azure Application object to be used for authentication with this keyvault.<br/>Minimum length = 1<br/>Maximum length = 63.

state¶

Current state of keyvault.<br/>Possible values = Created, Could not reach token endpoint, Authorization failed, Token parse error, Access token obtained, Keyvault host is reachable, Keyvault operation succeeded, Keyvault operation failed, Ops failed. Keyvault marked down.
