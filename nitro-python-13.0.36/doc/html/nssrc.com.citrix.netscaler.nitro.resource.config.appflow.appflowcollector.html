appflowcollector module¶

class appflowcollector¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for AppFlow collector resource.

name¶

Name for the collector. Must begin with an ASCII alphabetic or underscore (_) character, and must contain only ASCII alphanumeric, underscore, hash (#), period (.), space, colon (:), at (@), equals (=), and hyphen (-) characters. Only four collectors can be configured. The following requirement applies only to the Citrix ADC CLI: If the name includes one or more spaces, enclose the name in double or single quotation marks (for example, “my appflow collector” or ‘my appflow collector’).<br/>Minimum length = 1<br/>Maximum length = 127.

ipaddress¶

IPv4 address of the collector.

port¶

Port on which the collector listens.

netprofile¶

Netprofile to associate with the collector. The IP address defined in the profile is used as the source IP address for AppFlow traffic for this collector. If you do not set this parameter, the Citrix ADC IP (NSIP) address is used as the source IP address.<br/>Maximum length = 128.

transport¶

Type of collector: either logstream or ipfix or rest.<br/>Default value: ipfix,<br/>Possible values = ipfix, logstream, rest.

newname¶

New name for the collector. Must begin with an ASCII alphabetic or underscore (_) character, and must contain only ASCII alphanumeric, underscore, hash (#), period (.), space, colon (:), at(@), equals (=), and hyphen (-) characters. The following requirement applies only to the Citrix ADC CLI: If the name includes one or more spaces, enclose the name in double or single quotation marks (for example, “my appflow coll” or ‘my appflow coll’).<br/>Minimum length = 1.

state¶

Appflow collector state.<br/>Possible values = UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED.
