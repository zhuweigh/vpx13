nattype¶

Type of sessions to be displayed.<br/>Default value: NAT44<br/>Possible values = NAT44, DS-Lite, NAT64.

pcpsrcip¶

Internal IP address.

subscrip¶

IPV4 src address of Client in DS-Lite.

pcpsrcport¶

Internal Port.<br/>Range 1 - 65535<br/>* in CLI is represented as 65535 in NITRO API.

pcpdstip¶

Remote PEER IP address.

pcpdstport¶

Remote PEER Port.<br/>Range 1 - 65535<br/>* in CLI is represented as 65535 in NITRO API.

pcpnatip¶

External IP address.

pcpnatport¶

External Port Number.<br/>Range 1 - 65535<br/>* in CLI is represented as 65535 in NITRO API.

pcpprotocol¶

Protocol of the PCP mapping.<br/>Possible values = HopOpt, icmp, igmp, ggp, ipv4, st, tcp, udp.

pcpaddr¶

Address of PCP Map entity.

pcpnounce¶

PCP Mapping Nounce.

pcprefcnt¶

Refcount, which indicates how many session is active.

pcplifetime¶

Integer value that shows the requested lifetime.
