cloudparameter module¶

class cloudparameter¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for cloud parameter resource.

controllerfqdn¶

FQDN of the controller to which the Citrix ADC SDProxy Connects.<br/>Minimum length = 1.

controllerport¶

Port number of the controller to which the Citrix ADC SDProxy connects.<br/>Minimum length = 1<br/>Range 1 - 65535<br/>* in CLI is represented as 65535 in NITRO API.

instanceid¶

Instance ID of the customer provided by Trust.<br/>Minimum length = 1.

customerid¶

Customer ID of the citrix cloud customer.<br/>Minimum length = 1.

resourcelocation¶

Resource Location of the customer provided by Trust.<br/>Minimum length = 1.

activationcode¶

Activation code for the NGS Connector instance.

deployment¶

Describes if the customer is a Staging/Production or Dev Citrix Cloud customer.<br/>Possible values = , Production, Staging, Dev.

connectorresidence¶

Identifies whether the connector is located Onprem, Aws or Azure.<br/>Possible values = None, Onprem, Aws, Azure, Cpx.

controlconnectionstatus¶

Status of the control connection: In case of successful registration to the controller, connection status will be shown as “Registered” else “Unregistered”.
